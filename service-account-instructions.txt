# Firebase Service Account Instructions

To use Firebase Storage in this application, you need to add a Firebase service account key file.

## Steps to get your service account key:

1. Go to the Firebase Console: https://console.firebase.google.com/
2. Select your project (vertex-ai-436310)
3. Click on the gear icon (⚙️) next to "Project Overview" to open Project settings
4. Go to the "Service accounts" tab
5. Click "Generate new private key" button
6. Save the downloaded JSON file as `service-account.json` in the root directory of this project

## Important Security Notes:
- NEVER commit the service-account.json file to version control
- Add service-account.json to your .gitignore file
- Keep this file secure as it provides admin access to your Firebase project

## File Structure:
The service-account.json file should look something like this:

```json
{
  "type": "service_account",
  "project_id": "vertex-ai-436310",
  "private_key_id": "your-private-key-id",
  "private_key": "-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n",
  "client_email": "firebase-adminsdk-xxxx@vertex-ai-436310.iam.gserviceaccount.com",
  "client_id": "your-client-id",
  "auth_uri": "https://accounts.google.com/o/oauth2/auth",
  "token_uri": "https://oauth2.googleapis.com/token",
  "auth_provider_x509_cert_url": "https://www.googleapis.com/oauth2/v1/certs",
  "client_x509_cert_url": "https://www.googleapis.com/robot/v1/metadata/x509/firebase-adminsdk-xxxx%40vertex-ai-436310.iam.gserviceaccount.com",
  "universe_domain": "googleapis.com"
}
```
